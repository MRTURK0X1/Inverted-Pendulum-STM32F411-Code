# Inverted Pendulum Software for STM32F411-Blackpill dev Board
- Application is FreeRTOS based.
- Includes libraries for control system implementation ,and creation of dc motor and encoder objects. 
- Target to host communication pipeline is operational. Host to target communication is not implemented yet.
- Current application is developed for large scale projects in mind so it has significant ram and flash usage in favor of more streamlined programming experience for the programmer.
- A light weight version with less libraries and more direct coding will be uploaded as a separate repo.
